            Security Model: The hash is stored in your Apps Script, not in this platform.
            When you click Send in Module 2, your From Email is hashed and compared with the stored hash.
            No email is ever sent if the hashes don't match. The platform never sends emails directly.
          

        


        
        

      


      
      

        

          Follow these steps exactly. Each step takes about 30 seconds. Total setup time: ~5 minutes.
        


        

          
⚡ You are on Step 3 of the 5-step wizard

          
Complete these instructions to finish setup and unlock the Email Builder.

        


        

          

            
1

            

              
Check Compliance

              
Switch to the Compliance tab and verify all checks pass before proceeding. Export is blocked if any check fails.

            

          

          

            
2

            

              
Copy the Generated Script

              
Switch to the "Apps Script" tab and click Copy Script to copy the entire code to your clipboard.

            

          

          

            
3

            

              
Open Google Apps Script

              
Go to script.google.com and sign in with the exact same Gmail address you registered above.

            

          

          

            
4

            

              
Create a New Project

              
Click New Project (top left). Delete the placeholder code in the editor. Paste your copied script. Press Ctrl+S to save. Name it "Email Builder".

            

          

          

            
5

            

              
Deploy as Web App

              
Click Deploy → New Deployment. Set Type = Web App.

            

          

          

            
6

            

              
Configure Deployment Settings

              

                Set "Execute as" → Me (your Google account)

                Set "Who has access" → Anyone
              

            

          

          

            
7

            

              
Authorize Gmail Permissions

              
Click Deploy. Google will ask to authorize Gmail permissions. Click Allow. This lets your script send emails through your Gmail.

            

          

          

            
8

            

              
Copy Your Web App URL

              

                After deployment, you'll see a Web App URL:

                https://script.google.com/macros/s/AKfycb.../exec

                Copy this URL — you'll need it in Module 2.
              

            

          

          

            
9

            

              
Copy Your Script ID

              
In your Apps Script project URL bar, find the Script ID (the long string after /projects/). Copy it too.

            

          

          

            
10

            

              
Configure Module 2 & Test Connection

              

                Open the Email Builder, click ⚙ Account, and enter your:

                • From Email (the Gmail you registered)

                • Apps Script URL (from Step 8)

                • Script ID (from Step 9)

                Then click "Test Connection & Verify Ownership" to run all 5 verification phases.
              

            

          

          

            
11

            

              
🚀 You're Ready!

              
Once all verification phases pass and the Deployment Validation panel shows "Builder Ready", you can build email campaigns using the Bucket → Stack → Preview → Submit → Send workflow.
